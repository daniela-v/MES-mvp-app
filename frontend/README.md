# Frontend (React + Vite + TypeScript)

## Run locally
```bash
npm install
npm run dev
```

## Build and lint
```bash
npm run lint
npm run build
```

## Routes
- `/`
- `/course/:id`
- `/checkout/:courseId`
- `/onboarding?token=...`
- `/dashboard`
- `/enrolment/:enrolmentId`
- `/enrolment/:enrolmentId/lesson/:lessonId`
